What does it do?
================

Social Publisher is used to publish your typo3 pages into different social medias like Facebook, Twitter, Linkedin and Xing.
You can configure this easly, and also there is an option to test the configurations. To publish a page go to the edit page section  and you will find a tab named 'Social Post', here you will get the option to publish to social media. And also this tab contains the option to disable each social media for the particular page.
